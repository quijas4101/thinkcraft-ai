# ThinkCraft AI

A modern educational platform that combines AI-powered learning with project-based education.

## Features

- 🎓 Student Dashboard
- 👩‍🏫 Teacher Dashboard
- 🤖 AI-Powered Learning Assistant
- 📊 Progress Tracking
- 🎯 Challenges System
- 📂 Project Management

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Firebase
- Tailwind CSS
- Shadcn UI
